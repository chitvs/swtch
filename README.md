# sw

A minimal terminal stopwatch written in pure C.

```
[RUNNING] 00:00:12.345   laps: 3   [space] pause  [l] lap  [r] reset  [q] quit
```

## Build

```bash
make
```

## Install

```bash
sudo make install
```

## Usage

| Key     | Action        |
|---------|---------------|
| `space` | Pause / resume |
| `l`     | Record lap    |
| `r`     | Reset         |
| `q`     | Quit          |
